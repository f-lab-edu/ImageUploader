# Image Uploader

React 와 Typescript 를 사용하여 이미지를 업로드하고 업로드한 이미지를 리스트화 시키는 프로젝트입니다.

## 주요기능

### 1. 이미지 업로드 + 미리보기

- 이미지를 선택하면 미리보기 이미지가 나타납니다.

### 2. 이미지 등록

- 이미지를 등록하면 하단에 리스트로 나열합니다.
